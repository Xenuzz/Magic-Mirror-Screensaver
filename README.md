# Magic-Mirror-Screensaver

Dieses Magic Mirror Module ist dafür da einen ScreenSaver zu erstellen, wo sie ihre eigenen Bilder auswählen können. 


Config: 
modules: [
  {
    module: "MMM-ScreenSaver",
    config: {
      images: [
        "path/to/image1.jpg",
        "path/to/image2.png",
        "path/to/image3.gif"
      ]
    }
  }
]
